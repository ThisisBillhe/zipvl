ZIPVL: A Unified Framework for Dynamic Token Sparsification and KV Cache Compression.

Abstract

ZIPVL addresses the computational and memory bottlenecks of large vision-language models by introducing dynamic token sparsification and mixed-precision quantization of the KV cache.

Our approach adaptively adjusts the token sparsity ratio and applies mixed-precision quantization, achieving a 2.6Ã— speedup in the prefill phase and a 50% reduction in GPU memory usage with minimal accuracy loss.
